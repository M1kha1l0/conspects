# Указатели, массивы, строки
## Ссылка
У любой переменной любого типа данных есть определенная ячейка в памяти, в которой хранится значение. Оператор "&" позволяет узнать по какому адресу в памяти лежит переменная. Адрес не имеет типа данных, а также является константой, т.е. адрес переменной невозможно изменить.

Пример использования:
```cpp
#include <iostream>

int main() {
	int x = 10;
	std::cout << &x << '\n';
	// Примерный вывод: 0x7ffd0da7147c 
	return 0;
}
```

## Указатель
Указатель - переменная, диапазон значений, которой состоит из адресов ячеек памяти и специального значения (нулевого адреса).

Простыми словами:
Указатель указывает на ячейку памяти, к которой с его помощью мы можем обратиться. 
При инициализации указателя ему выдается *нулевой адрес (nullptr)*.  Это адрес зарезервированный под "ничего" и используется только для обозначения пустого указателя.
Указатель непосредственно хранит в себе адрес переменной в памяти, но также может работать и с объектами в памяти в помощью разыменования (звездочка).

```cpp
#include <iostream>

int main() {
	int x = 20;
	int *y = &x;
	int z = *y;
	
	// выведет 20, т.к. переменная хранит в себе число 20
	std::cout << "x = " << x << '\n';
	// выведет адрес в памяти x, т.к. указатель хранит адрес
	std::cout << "y = " << y << '\n';
	// выведет 20, т.к. разыменованный указатель возвращает значение в памяти
	std::cout << "z = " << z << '\n';
	
	return 0;
}
```

## Массив
Массив это набор чисел в памяти объединенные в один блок памяти. По сути своей массив это указатель на начала этого блока в памяти.
У массива, как и у указателя, есть тип данных (int, double, long, void). Тип данных в массиве позволяет компилятору определять на сколько сдвигать указатель в памяти для каждого индекса. Т.е. `&array[i]` = `&(array + sizeof(type))` 
Void указатель не имеет арифметики указателей и если пробегать по массиву типа `void*`, то мы будем бежать по байтам в массиве.

## Строки
Строки это те же массивы, но не произвольного типа, а `char*`. 
В конце каждой строки стоит специальный символ конца строки `\0`.
